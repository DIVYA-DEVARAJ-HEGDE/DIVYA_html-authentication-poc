# FarmConnect HTML Authentication Demo
This project demonstrates a basic authentication flow using only HTML pages and anchor tag navigation.

## Pages Included
- Login Page
- Registration Page
- Forgot Password Page
- Reset Password Page
- Dashboard Page

## How It Works
All pages are connected using anchor tags to simulate authentication flow.
No CSS or JavaScript is used as per assignment instructions.
